## 📋 프로젝트 소개

붕괴: 스타레일의 복잡한 계산 공식을 구현한 웹 애플리케이션입니다.  
캐릭터 스탯, 적 정보, 버프/디버프를 설정하여 각종 스탯, 최종 데미지 등을 계산할 수 있습니다.

## 🚀 주요 기능

- ⚔️ **실시간 데미지 계산**: 게임 내 공식 기반 정확한 계산
- 🎯 **캐릭터 커스터마이징**: 스탯, 스킬, 버프 설정
- 👹 **적 설정**: 레벨, 방어력, 약점 속성 조정
- 📊 **결과 분석**: 상세한 데미지 분해 정보
